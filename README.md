# 我们是谁
我们是面试助手小天使，希望能帮助到更多人在准备面试的道路上可以获得更多的收获，
同时我们还是字节跳动教育领域内的一群小二逼，我们充满理想，充满激情，一心想推动教育事业的发展

2019年7月，字节跳动教育业务部门成立。2020年10月29日，字节跳动高级副总裁、大力教育CEO陈林宣布独立品牌“大力教育”成立，字节跳动旗下所有的教育产品由大力教育承接。
目前，大力教育业务横跨Pre-K，K12，成人教育多年龄段，涵盖多学科、多课程，软硬件均有探索。旗下产品包括清北网校、GOGOKID、瓜瓜龙启蒙、开言英语、极课大数据、Ai学、教育硬件等。目前员工已经超过一万人。

具体到我们团队，是内容中台，从业务上来说是需求方和供给方的交易撮合平台，从技术上来说，是滴滴派单模式+12306的结合体，业务复杂度和技术挑战并存，对个人成长是非常有帮助的。
